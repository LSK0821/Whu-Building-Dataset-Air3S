# Whu-Building-Dataset-Air3S

A UAV visual localization dataset collected at Wuhan University using DJI Air3S.

## Overview

Whu-Building-Dataset-Air3S contains three UAV image datasets captured at Wuhan University:

- Whu-Huayuan
- Whu-Jiyuan
- Whu-Zhuoer

Images were captured using **DJI Air3S UAV** with onboard RGB camera **DJIFC9113**.

Flight altitude: **30–60 m**

Image resolution:

1024 × 768

Average GSD:

0.0497 m/pixel

The dataset is designed for evaluating:

- Visual localization
- Image retrieval
- SfM reconstruction
- UAV mapping
